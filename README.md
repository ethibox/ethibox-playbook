Ethibox ansible playbook
===

> Ethibox ansible playbook for debian/ubuntu

## Quick Start

```bash
git clone --recursive https://github.com/ethibox/ethibox-playbook
ansible-playbook playbook.yml -u <user> -i <host>,
```
